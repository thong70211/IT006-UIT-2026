# Nhật ký Bài tập Cá nhân

Đây là nơi lưu trữ các file bài tập trong quá trình học tập của tôi.

### Lưu ý về Bản quyền:
- Repository này chỉ phục vụ mục đích **lưu trữ nhật ký học tập** cá nhân.
- Tôi **không sở hữu** bản quyền đối với các tài liệu, đề bài hoặc tài nguyên học thuật có trong đây (trừ phần lời giải do tôi tự thực hiện).
- Tất cả tài nguyên thuộc quyền sở hữu của tác giả/giảng viên hoặc tổ chức giáo dục tương ứng.

---
*Mục đích: Theo dõi tiến độ học tập cá nhân.*
